# Changelog

## [1.0.0] - 2025-03-07

### Added

- **content-planner**: 选题库构建、编辑日历、排期管理
- **article-generator**: 技术博客生成（Markdown/Obsidian 格式），Gemini AI 图片生成，PicGo 图床上传
- **content-reviewer**: 7 维评分审查（可读性、逻辑流、标题吸引力、事实准确性、AI 痕迹、平台适配、原创性），快速/完整模式
- **wechat-seo-optimizer**: 标题 A/B 测试、关键词策略、摘要优化、封面图建议
- **wechat-article-converter**: Markdown → 微信 HTML（7 Python 主题 + 9 Go 后端主题），预览服务器，草稿箱上传
- **content-remixer**: 爆款文章拆解 → 创意积木提取 → 组装新内容
- **content-pipeline agent**: 端到端流水线编排
- 统一配置系统（`config/config.json` + 环境变量 + `~/.claude/env.json` 向后兼容）
